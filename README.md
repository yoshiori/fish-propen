# propen

propen is pull-request page open command on fish-shell.


## Installation

```sh
$ omf install https://github.com/yoshiori/fish-propen
```
## Usage

## Open `origin` remote

```sh
$ propen
```

## Open other remote

```sh
$ git remote
mine
origin
$ propen mine
```
